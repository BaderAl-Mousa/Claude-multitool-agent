# Claude Multi-Tool Agent

This project demonstrates building a multi-turn AI system using Claude API.

## Main Features
- Tool use with Claude
- Multi-turn conversations
- Multiple tool integration

## Experiments
- Fine-grained tool calling
- Text editing tool
- Web search tool

## Setup
1. Install requirements
2. Add your API key in a .env file
3. Run the project

## Notes
This project was built as part of learning Claude API and AI systems.
